• Position brightness display
    • When you start brightness the first time, sliders are being displayed at the center of the screen
    • Click onto the name of the 1st slider and drag the slider to the desired position
    • Click with the right mouse button on to open the settings
    • Unselect Adjust display position of sliders
  • Manage settings
    • Click with the right mouse button on to open the settings
    • Change Language and Look And Feel to your needs
    • Select Dark Mode Icons if your are using a dark theme
    • Adjust display position of sliders. See above
    • Select Link Sliders to change the brightness of all displays at once
    • Color temperature
      • automatically: Color temperature is switched according to sunrise and sunset.
        Sunrise and sunset are calculated using the geocoordinates. You can find geocoordinates
        for your place of residence e.g. at https://www.latlong.net/.
      • manually: Night mode is active for the specified period of time
  • Troubleshooting
    • If the tray icon looks like an unexpected error occured. Please restart brightness
  • Logging
    • By default brightness.log is generated in the home diretory of the current user. The file is hidden
    • To change the location, use -DlogDir=<your_log_folder>
    • To change the log level, use -DlogLevel=<new log level>
    when starting brightness
